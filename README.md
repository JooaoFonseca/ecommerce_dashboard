Apresenta a criação de um Dashboard de Análise de Dados para um e-commerce, utilizando a ferramenta Power BI para transformar dados brutos em insights de negócios. Esta parte foca na visualização e análise.

Seção do Dashboard | Funcionalidades Desenvolvidas
Cálculo de KPIs -> "Criação de DAX Measures (medidas e cálculos) para determinar Key Performance Indicators (KPIs) como Vendas (Sales), Lucro (Profit) e Quantidade Vendida (Quantity)."

Análise de Tendências -> Implementação de indicadores de crescimento/queda ano a ano (Year-on-Year Growth) e visualização de tendências.

Gráficos e Visualizações -> "Criação de diversos gráficos para analisar o desempenho: Vendas por Categoria, Vendas por Estado/Região, Produtos mais vendidos (Top 5) e menos vendidos (Bottom 5), e Vendas por Tipo de Envio (Shipping Type)."

Filtros Interativos -> "Configuração de Slicers (segmentadores) e filtros interativos para permitir que o usuário explore os dados por segmento (Segment) ou outras variáveis, garantindo que o dashboard seja dinâmico."

Foco -> Do tratamento inicial dos dados (Data Cleaning/Modelling) até a apresentação final de um dashboard interativo para tomada de decisões.
